```

### 2. استخراج قوائم المنتجات

```
Before:
┌─────────────────┐
│ Product 1       │

├─────────────────┤
│ Product 2       │
│ $49.99          │
└─────────────────┘

After:
[
  {"name": "Product 1", "price": "$99.99"},
  {"name": "Product 2", "price": "$49.99"}
]
```

```
Before:

│ Name │ Grade │ Status │
